# Singly Linked List Demo

Run the following in Cygwin (assuming that you've got Git and g++ installed correctly to locally clone this repository and run this demo code):

```bash
~$ git clone https://github.com/BryceTuppurainen/PASS-LinkedListDemo

~$ g++ LinkedListDemo.cpp -std=c++98 -o LinkedListDemo.exe

~$ ./LinkedListDemo.exe
```
